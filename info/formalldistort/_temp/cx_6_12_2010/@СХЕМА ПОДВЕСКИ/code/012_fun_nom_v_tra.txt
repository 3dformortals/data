function nom_v_tra(L,X,Y)

nom_v_tra = (Y-1)*L+X

end function

определение номера элемента в последовательном массиве tra содержащем трассу, по двум координатам поля точек трассы
L - длина трассы вдоль оси X в колличестве точек [mm]
X - координата X точки трассы
Y - координата Y точки трассы
