pngbatchpixelplus (18-12-2016)
требуется библиотека pillow и python
тестировалась под python 3.4.4, pillow 3.4.2
ПО ОЧЕРЕДИ ПЕРЕБИРАЕТ ВСЕ PNG В ПАПКЕ ГДЕ НАХОДИТСЯ
ФАЙЛ ПРОГРАМЫ
помещаем заготовку png в папку с файлом програмы.
в заготовке убираем весь фон...например белый делаем прозрачным
и прога разудавает каждую непрозрачную, альфа больше 0,
точку(пиксель) оригинала до 9 точек(пикселей),
 рамка в один пиксель по краям рисунка не проверяется алгоритмом...
 все точки результата заданы цветом RGBA 255...
на входе принимает только png
большой файл подвешивает консоль и после нажатия (input) enter
окно не исчезает... на крестик красный закрывать
