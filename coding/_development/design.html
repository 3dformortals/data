draw liquid portal 100x100 px:
-create new file. Set unit to px, scale to 1, page size to 100x100, save file
-create round 500x500 px, fill rgb(0,79,255), save file
-Filters - Bumps - Plastify (фильтры - выпуклости - пластификация)
-Edit - Make a Bitmap Copy (правка - сделать растровую копию)
-select bitmap copy
-Object - Transform - Scale - Scale proportionally - 20% (объект - трансформировать - масштаб - пропорциональное масштабирование - 20%)
-Filters - Materials - Lizard Skin (фильтры - материалы - кожа ящерицы)
-Filters - Bevels - Neon (фильтры - фаска - неон)
-Filters - Blurs - Clean Edges - 2.0 (фильтры - размывание - чистые края - 2.0)
result получается как гель рыхлый

result получается чтото вроде камня
